Axios adalah library JavaScript populer yang digunakan untuk membuat HTTP requests dari browser atau Node.js. Axios menyederhanakan proses pengiriman asynchronous HTTP requests ke server dan menangani responsnya. Axios mendukung fitur seperti interceptors, menangani requests dan response header, dan menangani berbagai tipe data, seperti JSON.
